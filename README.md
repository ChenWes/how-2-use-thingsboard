### Thingsboard使用
基于**3.2**分支的使用文档，包含基础、高阶、二开和扩展

### 准备
- Jdk，11+（因3.2.2版本需要11），用于编译运行
- Maven，3.2.1+，不强制需要，某些IDE也自带
- IDE，推荐Idea，查看代码和运行
- Html5浏览器，推荐使用Chorme，用于页面操作
- 容器引擎，推荐使用Docker，用于初始化环境，比如PG数据库
- Mqtt客户端，推荐使用MqttBox，用于模拟时序数据
